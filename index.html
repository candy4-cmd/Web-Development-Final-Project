Beginner's Web Development Journey!

Learning Path

To become a web developer, you first learn HTML. HTML is the standard markup language for web pages. With HTML you can create your own website.

Then you learn CSS, a style sheet language used for specifying the presentation and styling of a document written in HTML.

Finally, you learn JavaScript, a programming language and core web technology. It powers interactive behavior on 99% of websites!

Some basic HTML elements include: <head>, <title>, <body>, <header>, <footer>, <article>, <section>, <p>, <div>.
